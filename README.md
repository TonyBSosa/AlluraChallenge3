# 📊 Análisis de Ventas de Tiendas

Este proyecto consiste en un análisis exploratorio de datos (EDA) realizado en Python utilizando Google Colab. El objetivo del análisis es evaluar el desempeño de cuatro tiendas diferentes a partir de sus datos de ventas y determinar cuál de ellas presenta el menor rendimiento para recomendar su cierre.

El análisis se basa en distintos indicadores como ingresos totales, categorías de productos más vendidas, calificaciones de los clientes y costos de envío.

---

# 📁 Dataset

El conjunto de datos incluye información sobre:

- Producto vendido
- Categoría del producto
- Precio
- Costo de envío
- Calificación del cliente
- Ubicación de compra
- Método de pago

Cada tienda cuenta con su propio dataset, lo que permite comparar su desempeño.

---

# 🔎 Análisis Realizados

Durante el proyecto se realizaron los siguientes análisis:

### 1️⃣ Ingreso total por tienda
Se calcularon los ingresos totales sumando el precio de todos los productos vendidos en cada tienda.

### 2️⃣ Ventas por categoría
Se analizaron las categorías de productos más vendidas para identificar cuáles generan mayor demanda.

### 3️⃣ Calificación promedio de clientes
Se calcularon las valoraciones promedio para evaluar la satisfacción del cliente en cada tienda.

### 4️⃣ Productos más y menos vendidos
Se identificaron los productos con mayor y menor volumen de ventas.

### 5️⃣ Costo promedio de envío
Se calculó el costo promedio de envío en cada tienda para analizar su impacto potencial en las ventas.

---

# 📊 Visualizaciones

Se generaron diferentes gráficos utilizando **Matplotlib** para facilitar la interpretación de los resultados:

- Gráfico de ingresos por tienda
- Distribución de ventas por categoría
- Comparación de calificaciones promedio
- Comparación de costos de envío

Estas visualizaciones permiten entender rápidamente las diferencias entre las tiendas.

---

# 📌 Conclusión

Después de analizar los datos, se observa que **la Tienda 1 presenta el mayor volumen de ingresos**, lo que indica una mayor actividad comercial.

Por otro lado, **la Tienda 4 presenta el menor nivel de ingresos**, a pesar de tener el costo de envío promedio más bajo. Esto sugiere que su desempeño general es inferior al de las otras tiendas.

Con base en los resultados obtenidos, la recomendación sería **cerrar la Tienda 4**, ya que muestra el menor rendimiento dentro del conjunto analizado.
